## PPT

如果想要查看ppt讲稿，依次执行以下操作：
```javascript
npm install -g nodeppt
````

在本项目根目录下：
```javascript
nodeppt start
````
浏览器会自动打开，选择ppt即可观看。比如选择nodejs.md即可观看nodejs的PPT内容。


## 实例

可以直接在浏览器中打开example/shoppingCart/shoppngCart.html文件查看demo。

也可以在example目录下执行：
```javascript
grunt
````
